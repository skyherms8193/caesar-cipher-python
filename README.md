# Caesar Cipher in Python

This project was built as part of the freeCodeCamp Python curriculum.

It implements a Caesar cipher with separate encryption and decryption functions, along with validation for shift values.

## Features
- Encrypts text using a Caesar cipher
- Decrypts text using the same shift value
- Preserves uppercase and lowercase letters
- Validates that shift is an integer between 1 and 25

## Files
- `caesar.py` — main Python implementation

## Example
```python
print(encrypt("Hello", 3))   # Khoor
print(decrypt("Khoor", 3))   # Hello
```
